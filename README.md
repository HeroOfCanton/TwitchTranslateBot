# TwitchTranslateBot

The goal of this bot is to have a real-time translation of non-English chat messages.

This will be accomplished by:

1- Listener in live Twitch chat for any chat messages not in English

2- When identified, send them to Google Translate to translate them into English

3- Post the translated version back in chat.

Very much a work in progress, with no guarantees when this will be finished.
